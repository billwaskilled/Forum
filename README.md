<h1 align="center">常英杰开发的一个论坛</h1>
<h2 align="center"> 基于Spring+Spring MVC+Mybatis(Maven方式构建)</h2>


## 说明
1. 该项目为练手项目，其中难免有很多不足之处，如有发现，还请不吝赐教

## 如何使用
```aidl
$ git clone https://github.com/billwaskilled/forum

$ cd forum

$ mvn clean compile

$ mvn clean package

$ mvn clean install

$ mvn jetty:run
 
 http://localhost:8080
```
